사용 방법

  1. PC방 프로그램에 표시된 시작 전 남은 시간을 입력합니다.
  2. 플레이할 유료게임을 선택하고 유료게임 시작 버튼을 누릅니다.
  3. 게임이 끝난 뒤 PC방 프로그램의 종료 후 남은 시간을 입력합니다.
  4. 분석하기 버튼을 누르면 차감 시간, 차감 배율, 위험도가 계산됩니다.
  5. 분석 기록은 브라우저에 저장되며, CSV 파일로 다운로드할 수 있습니다.

서비스 설명

이 서비스는 PC방 시스템과 직접 연동하지 않고, 사용자가 확인한 남은 시간을 기준으로 유료게임 이용 중 시간이 얼마나 빠르게 차감됐는지 역산합니다.

계산식: 차감 배율 = PC방에서 차감된 시간 ÷ 실제 플레이 시간
