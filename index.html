PC방 유료게임 차감 분석기

유료게임을 플레이한 뒤, PC방 시간이 실제로 얼마나 더 빠르게 차감됐는지 분석해드립니다.

유료게임 시작 후 분석 결과가 여기에 표시됩니다.

사용 방법

  1. PC방 프로그램에 표시된 남은 시간을 입력합니다.
  2. 유료게임을 시작할 때 유료게임 시작 버튼을 누릅니다.
  3. 게임이 끝난 뒤 PC방 프로그램의 남은 시간을 다시 입력합니다.
  4. 분석하기 버튼을 누르면 실제 차감 배율을 확인할 수 있습니다.
