# PC-room-service
# PC방 유료게임 차감 분석기

PC방에서 유료게임을 이용할 때 시간이 실제로 얼마나 빠르게 차감되는지 분석하는 웹 서비스입니다.

## 프로젝트 소개

일부 PC방에서는 특정 유료게임을 실행하면 일반 사용 시간보다 시간이 더 빠르게 차감될 수 있습니다.
하지만 사용자는 실제로 몇 분이 추가로 차감되었는지, 어느 정도 빠르게 시간이 줄어들었는지 바로 알기 어렵습니다.

이 프로젝트는 사용자가 직접 확인한 **시작 전 남은 시간**과 **종료 후 남은 시간**을 비교하여, 유료게임 이용 중 PC방 시간이 얼마나 빠르게 차감되었는지 계산합니다.

## 개발 목적

이 서비스의 목적은 PC방 이용자가 유료게임 플레이 후 실제 차감된 시간을 쉽게 확인할 수 있도록 돕는 것입니다.

사용자는 유료게임 시작 전 남은 시간과 종료 후 남은 시간을 입력하면 다음 정보를 확인할 수 있습니다.

* 실제 플레이 시간
* PC방에서 실제로 차감된 시간
* 추가로 차감된 시간
* 차감 배율
* 추가 차감률
* 차감 위험도

## 주요 기능

### 1. 유료게임 시간 측정

사용자가 유료게임을 시작할 때 버튼을 누르면 시작 시각이 자동으로 저장됩니다.

### 2. 종료 후 시간 분석

게임이 끝난 뒤 PC방 프로그램에 표시된 남은 시간을 입력하면, 시작 전 시간과 비교하여 실제 차감 시간을 계산합니다.

### 3. 차감 배율 계산

실제 플레이 시간과 PC방 차감 시간을 비교하여 시간이 몇 배 빠르게 줄어들었는지 계산합니다.

### 4. 위험도 표시

차감 배율에 따라 다음과 같은 상태를 표시합니다.

* 정상
* 주의
* 빠른 차감
* 매우 빠른 차감

### 5. 분석 기록 저장

분석 결과는 브라우저 LocalStorage에 저장됩니다.
새로고침해도 최근 분석 기록을 확인할 수 있습니다.

### 6. 게임별 통계

게임별 평균 차감 배율을 계산하여 어떤 게임에서 시간이 더 빠르게 차감되었는지 확인할 수 있습니다.

### 7. CSV 다운로드

분석 기록을 CSV 파일로 다운로드할 수 있습니다.

### 8. 기록 관리

분석 기록을 개별 삭제하거나 전체 삭제할 수 있습니다.

## 사용 방법

1. PC방 프로그램에 표시된 시작 전 남은 시간을 입력합니다.
2. 플레이할 유료게임을 선택합니다.
3. **유료게임 시작** 버튼을 누릅니다.
4. 게임을 플레이합니다.
5. 게임 종료 후 PC방 프로그램에 표시된 남은 시간을 입력합니다.
6. **분석하기** 버튼을 누릅니다.
7. 분석 결과와 최근 기록, 누적 통계를 확인합니다.

## 계산 방식

```text
실제 플레이 시간 = 종료 시각 - 시작 시각
PC방 차감 시간 = 시작 전 남은 시간 - 종료 후 남은 시간
추가 차감 시간 = PC방 차감 시간 - 실제 플레이 시간
차감 배율 = PC방 차감 시간 ÷ 실제 플레이 시간
추가 차감률 = (차감 배율 - 1) × 100
```

## 예시

시작 전 남은 시간이 3시간이고, 실제로 30분 동안 유료게임을 플레이한 뒤 종료 후 남은 시간이 2시간 15분이라면 다음과 같이 계산됩니다.

```text
시작 전 남은 시간: 180분
종료 후 남은 시간: 135분
PC방 차감 시간: 45분
실제 플레이 시간: 30분
차감 배율: 1.5배
추가 차감 시간: 15분
```

즉, 사용자는 30분 동안 플레이했지만 PC방 시간은 45분이 차감된 것입니다.

## 프로젝트 특징

이 프로젝트는 별도의 서버나 데이터베이스 없이 브라우저에서 작동합니다.
분석 기록은 LocalStorage를 사용하여 사용자의 브라우저에 저장됩니다.

## 한계점

이 서비스는 PC방 관리 프로그램과 직접 연동하지 않습니다.
따라서 사용자가 직접 확인한 남은 시간을 기준으로 계산합니다.

또한 PC방마다 요금 정책이나 유료게임 차감 방식이 다를 수 있으므로, 계산 결과는 사용자가 입력한 값에 기반한 분석 결과입니다.
